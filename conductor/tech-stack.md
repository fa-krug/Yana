# Tech Stack

## Core Technologies
- **Language:** TypeScript
- **Frontend Framework:** Angular (v21+) with Server-Side Rendering (SSR)
- **Backend Framework:** Express (v5+) and tRPC
- **Database:** SQLite (using `better-sqlite3`) with Drizzle ORM
- **API Communication:** tRPC for type-safe communication between frontend and backend.

## Tooling and Infrastructure
- **Build Tool:** Angular CLI
- **Testing:** Vitest for unit and integration testing.
- **Linting:** ESLint with Angular and TypeScript specific plugins.
- **Formatting:** Prettier
- **Task Runner:** npm scripts
- **Version Control:** Git
- **Documentation:** Markdown

## Key Libraries
- **Content Processing:** Cheerio, rss-parser, marked
- **Data Validation:** Zod
- **Utilities:** date-fns, axios, pino (logging)
