Setup Instructions

To run the Angular frontend, follow these steps:

1. Build the Angular Application

cd frontend
ng build --configuration production

2. Copy index.html to Templates

cp dist/browser/index.html ../templates/

3. Collect Static Files

cd ..
python manage.py collectstatic --noinput

4. Run the Development Server

python manage.py runserver

Development Mode

For development with hot reload, run the Angular dev server separately:

cd frontend
npm install
ng serve

Then access the app at http://localhost:4200
